# 简历视觉设计规范

本规范用于新增或调整模板。目标是让简历在 A4 PDF、离线环境和不同机器上保持稳定，
并确保所有内置模板采用本项目独立实现的结构与样式。

## 1. 稳定渲染原则

- 内容只维护在 `resume.md`，模板只负责排版。
- 使用 Jinja 生成 HTML，使用 WeasyPrint 输出 PDF。
- 不加载 CDN、Google Fonts、网页图标库或远程图片。
- 优先使用正常文档流、table 和简单 block；谨慎使用复杂 grid、绝对定位和大面积背景。
- 经历条目使用 `break-inside: avoid`，但不要对整个大章节禁止分页。
- 所有模板必须在无照片和有照片两种情况下正常排版。

共享离线基线位于 `assets/styles/resume-base.css`，模板 CSS 只覆盖自身差异。

## 2. 字体

### 无衬线字体

适用于技术、产品、互联网岗位，也是默认字体：

```css
"Hiragino Sans GB", "Microsoft YaHei", "Noto Sans CJK SC",
"Source Han Sans SC", "Heiti SC", sans-serif
```

### 衬线字体

只用于正式、编辑感较强的模板，不作为默认方案：

```css
"Songti SC", "STSong", "Noto Serif CJK SC",
"Source Han Serif SC", serif
```

不要仅指定某一种商业字体。找不到首选字体时必须有系统字体回退。

## 3. A4 尺度与字号

| 元素 | 推荐值 | 可接受范围 |
|---|---:|---:|
| 页边距 | 上下 12–14mm，左右 15–16mm | 10–18mm |
| 正文 | 10pt | 9.5–10.5pt |
| 正文行距 | 1.48 | 1.4–1.6 |
| 姓名 | 21–22pt | 20–24pt |
| 求职意向 | 10.5–11pt | 10–12pt |
| 一级模块标题 | 11.5–12pt | 11–13pt |
| 经历标题 | 10.4–10.8pt | 10–11.5pt |
| 日期/地点 | 8.9–9.5pt | 8.5–10pt |
| 模块间距 | 8–11px | 6–14px |
| 条目间距 | 5–8px | 4–10px |

校招一页简历不要通过把正文压到 9pt 以下解决溢出，应先删减低价值内容。

## 4. 颜色与线条

- 正文：`#222` 或 `#2a2a2a`。
- 次要信息：`#555`–`#777`。
- 分隔线：`#d7d7d7`–`#ededed`。
- 主色只用于姓名、模块标题、细线和项目符号，避免大面积高饱和背景。
- 黑白打印时仍须保持清晰层级，不能只依靠颜色区分。
- 主色使用 `--accent`，允许由 `--accent` 命令覆盖。

## 5. 照片

- 默认不放照片；用户明确需要时再使用 `photo:`。
- 建议比例约 3:4，尺寸 25–32mm × 33–42mm。
- 使用 `object-fit: cover`，不要拉伸。
- 照片不能挤压姓名、电话和邮箱；内容较多时优先选择无照片版本。

## 6. 内置版式类型

1. **Markdown 紧凑单栏**：居中或左右分布页眉、细分隔线、小节连续排列。稳定性最高。
2. **经典单栏**：黑白、模块标题下划线、经历标题和日期左右对齐。ATS 最稳。
3. **侧栏式**：左侧联系方式/技能，右侧经历。视觉突出，但跨页和内容密度更敏感。
4. **时间线式**：使用竖轴组织经历。适合条目多但每条较短的简历。
5. **极简式**：大留白、低对比细线。适合内容少且经过充分精简的候选人。
6. **色带标题式**：浅色模块底纹或左侧竖线。适合中文校招简历。
7. **身份栏式**：页首左侧姓名身份、右侧求职信息，后续内容通栏。适合正式或编辑感版式。

新增模板必须独立实现，不得引入第三方作者信息、示例内容、图标文件、装饰水印或专有视觉组合。

## 7. 当前模板参数

| 模板 | 正文 | 姓名 | 标题 | 页边距 | 照片 |
|---|---:|---:|---:|---|---|
| `compact` | 10pt | 21pt | 11.5pt | 12mm / 15mm | 25×33mm |
| `classic` | 10.5pt | 22pt | 12pt | 14mm / 16mm | 约 20×26mm |
| `modern` | 10.5pt | 20pt | 12pt | 12mm / 侧栏通栏 | 32×42mm |
| `timeline` | 9.8pt | 21pt | 12pt | 11mm / 14mm | 不支持 |
| `minimal` | 9.8pt | 22pt | 10pt | 13mm / 16mm | 约 19×24mm |

## 8. 新模板验收

- 示例简历能生成 A4 PDF，中文不乱码。
- 姓名、电话、邮箱、日期不会重叠或越界。
- 1 页与 2 页输入都能自然分页。
- 自定义主色只改变装饰层级，不降低正文对比度。
- 移除 `photo` 字段后不留下空白占位。
- PDF 文本可复制、可搜索，不把正文转成图片。
