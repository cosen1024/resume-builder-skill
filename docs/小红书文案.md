# 小红书文案

## 标题

别再用Word改简历了！AI一键生成5套模板PDF

## 封面图建议

5 套模板（compact / classic / modern / timeline / minimal）的 PDF 截图横向拼在一起，底部标注模板名称，顶部大字："一行命令，5 套模板随便换"

## 正文

写简历最痛苦的不是写内容，是排版改到第 20 版还是丑。
Word 对不齐、在线工具导出要会员、AI 改简历改完还得自己排版……
所以我自己用 Claude Code 做了一个简历 Skill 🔧

它不只是帮你"写"简历，而是直接帮你"做"简历——从内容润色到 A4 PDF 成品，一步到位。

——

✅ **直接出 PDF**
不是给你一堆文字让你自己排版，是直接渲染成 A4 PDF，打开就能投。离线生成，不吃网速不要会员。

📌 **5 套专业模板随便切**
同一份简历内容，换模板一行命令：
🔹 compact — 紧凑单栏，技术岗首选，内容多也能压一页
🔹 classic — 黑白严肃，ATS 解析最稳，海投/国企/银行闭眼选
🔹 modern — 左侧栏+彩色标题，互联网/产品岗看这个
🔹 timeline — 时间线布局，实习/项目经历多的同学很出彩
🔹 minimal — 极简大留白，内容精炼、想要设计感的选它

🖼️ **头像可放也可不放**
国内岗位需要照片时，在 Markdown 里加一行 `photo` 就能自动裁切排版；
compact / classic / modern / minimal 支持头像，删掉字段就是无头像版本。

🎨 **7 种配色 + 自定义**
蓝 / 青绿 / 酒红 / 墨黑 / 紫 / 绿 / 焦橙，或者直接传色值。
同一个模板换个颜色完全不同的气质。

📝 **AI 自动润色**
用 STAR 句式帮你改写每条经历，把流水账变成有数据的成果：
❌ 负责后端 API 的开发与维护
✅ 优化任务调度服务，引入本地缓存与批量写入，P99 延迟从 640ms → 210ms，支撑日均 920 万次请求

这不是 AI 编的，是从你真实经历里追问挖出来的数字。

🎯 **JD 匹配分析**
贴一份目标岗位的 JD，自动告诉你：
· 命中了哪些关键词 ✓
· 缺失了哪些关键词 ✗
· 哪些经历该上移、加重
帮你从"一份简历投所有"变成"每份都有针对性"。

📎 **多种输入方式**
有表格？CSV / xlsx 直接导入。
有旧简历？Markdown 丢进去润色。
什么都没有？对话式采集，AI 会追问你背景、做了什么、结果怎么样，帮你把经历挖出来。

——

⚡ 使用方式超简单：
安装好 Skill 后，一句话调用👇
"使用 $resume-builder，根据我的经历生成一份后端校招简历"
"使用 $resume-builder，帮我润色这份简历并渲染成 compact 模板"
"使用 $resume-builder，根据这份 JD 调整我的简历"

支持 Claude Code 和 Codex，prompt 复制到其他模型也能用内容润色的部分，但模板渲染需要在 Claude Code / Codex 里跑 🫡

——

做这个 Skill 的原因很简单：
市面上的 AI 简历工具，要么只帮你写内容不管排版，要么模板丑还收费，要么压根不适合中文求职场景。

我想要的是——写完内容直接出成品，模板好看、配色能换、一页纸搞定，个人非商业使用也没有会员门槛。

找不到，就自己做了一个 ✊
